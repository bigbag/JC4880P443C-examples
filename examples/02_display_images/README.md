# 02 Display Images

Graphics rendering demonstration with various visual elements.

## Description

Showcases LVGL's graphics capabilities through multiple demo modes, demonstrating colors, gradients, shapes, animations, and text rendering.

## Features

- Five switchable demo modes via "Next" button
- Color bars (RGB test pattern with 8 colors)
- Gradient rendering (smooth red-to-blue transition)
- Shape drawing (circles, rectangles, rings)
- Animated spinner
- Text styles with multiple font sizes and colors

## Demo Modes

1. **Color Bars** - 8-color test pattern
2. **Gradient** - Smooth color transition
3. **Shapes** - Circle, rounded rectangle, square with border, ring
4. **Animation** - Spinning loader animation
5. **Text** - Multiple font sizes and colors

## Build and Flash

```bash
pio run -t upload
```
